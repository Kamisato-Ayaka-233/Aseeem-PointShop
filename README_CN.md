# Aseeem PointShop
## Aseeem 点数商店

> 这个项目目前还处于早期阶段，我还没有充分测试它。  
> 可能会有些Bug，请谨慎使用。

这是一个新的Gmod点数商店，我叫它“Aseeem 点数商店”。

## 特性

这很像“点数商店2”插件，不过因为那个破商店太##难用了……

* 完成了玩家购买、卖出、装备和取消装备等基本功能。
* 支持一次性物品。
* 3种物品图标类型（图片、文本和模型）
* 现在只是支持了玩家尾迹的功能（因为做这个最简单）
* 完整的中文本地化和“*残废*”的英文（XD）
* 现在的所有数据都保存在JSON文件中。

我只做了它5天，还有许多许多的功能没做完。
